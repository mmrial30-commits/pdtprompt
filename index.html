1번에는 형태를 절대 바꾸면 안 되는 기준 팔찌 사진, 2번에는 적용할 원석 사진을 넣습니다. 아래 프롬프트는 새 팔찌를 생성하는 방식이 아니라 1번 사진의 기존 비즈 안에 원석 재질만 덮어씌우는 편집 방식으로 강화했습니다.

착용컷 STEP 1 · B 만들기

원석 사진을 Claude로 분석해 ‘B 원석 설명’을 만들기

중요: 아래의 긴 문장은 착용컷 생성용 B 프롬프트 자체가 아닙니다. Claude에게 원석 사진을 분석시키기 위한 분석 명령문입니다.

전체 흐름
① Claude에 원석 사진 첨부 → ② 아래 분석 명령문 복사·전송 → ③ Claude가 출력한 영어 분석 결과 복사 → ④ 그 결과를 아래의 ‘B 원석에 대한 설명’ 입력칸에 붙여넣기
1Claude 새 채팅 열기 ↗
2분석할 원석 사진 첨부
3분석 명령문 붙여넣기
4출력된 영어 결과 복사

B-1. Claude에 넣는 원석 분석 명령문

원석 사진과 함께 Claude에 넣습니다. Claude가 사진의 색상, 광택, 투명도, 내포물과 광학 효과를 영어로 정리해 줍니다.

B-2. Claude가 만든 결과를 여기에 붙여넣기

아래 입력칸에는 분석 명령문을 넣는 것이 아니라 Claude의 영어 출력 결과를 붙여넣습니다. 이 내용이 A 공통 프롬프트와 C 포즈 프롬프트 사이에 자동으로 들어갑니다.

👇 Claude가 출력한 영어 분석 결과 전체를 아래 칸에 그대로 붙여넣으세요.

포즈 선택하기

코드나 한글 키워드로 검색한 뒤 원하는 카드를 클릭하세요. 예: 손목, 주머니, 청바지, 항공, 클로즈업.

아직 선택된 포즈가 없습니다.
착용컷 STEP 3

최종 프롬프트 조합기

생성된 최종 프롬프트로 이동

Claude 출력 결과를 붙여넣고 포즈를 선택하면 최종 착용컷 프롬프트가 자동으로 만들어집니다.

자동 조합 순서
팔찌 기본 규칙 + Claude 원석 설명 + 선택한 포즈 + 추가 수정사항
펜던트 생성 방식 선택

펜던트가 계속 틀어지거나 누락되면 먼저 ‘펜던트 없이’로 안정적인 착용컷을 만든 뒤 별도 합성에 사용하세요.
